\# Git шпаргалка для друзей



\## Что такое Git?

Git помогает сохранять историю изменений в коде и работать над проектом вместе с другими.



\## Главные команды (которые нужны каждый день)



\*\*git status\*\* — проверить, что изменилось



\*\*git add .\*\* — подготовить все изменения к сохранению



\*\*git commit -m "что сделал"\*\* — сохранить изменения



\*\*git push\*\* — отправить на GitHub



\## Один раз сделать (при начале работы)



\*\*git init\*\* — начать новый проект



\*\*git remote add origin ссылка\*\* — связать с GitHub



\*\*git clone ссылка\*\* — скачать чужой проект



\## Как работать (коротко)



1\. Изменил файлы

2\. `git add .`

3\. `git commit -m "сделал то-то"`

4\. `git push`
